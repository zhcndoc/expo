[Expo Application Services (EAS)](/eas/) 提供 [免费访问](https://expo.dev/eas/fair-use#commercial-usage) 有限数量的低优先级构建，适用于 [EAS Build](/build/introduction/)，以及通过 [EAS Update](/eas-update/introduction/) 提供的免费更新。这些限制每月重置。

除免费套餐外，还有不同的订阅方案以满足各类客户及其需求。每个付费方案都提供积分，使 EAS Build 能够进行优先级构建，并通过更多的月活跃用户和额外带宽，为 EAS Update 提供更广泛的访问。我们还提供可补充订阅的附加项，并启用可选功能，以满足客户的更多需求。

本页列出了不同的订阅方案及可用的附加项。

## 订阅

订阅按月计费，全球价格一致（未含税）。要查看您账户当前的订阅方案，请前往[您的账户计费页面](https://expo.dev/settings/billing)，在 **Current Plan** 下即可查看当前方案的详细信息。

我们还可根据需要提供年度合同。请联系我们的[客户支持](https://expo.dev/contact)团队，了解年度方案是否适合您。

## 方案

每个方案都有特定限制。不过，订阅者可以超出这些限制，并通过[按使用量计费](/billing/usage-based-pricing/)为额外使用付费。

### Production

Production 方案专为专业开发者和小型企业设计。订阅此方案可获得：

- 可靠的生产级服务
- 用于 EAS Build 高优先级构建的每月[积分](/billing/usage-based-pricing/)
- 更多唯一用户、更高带宽以及更大的 EAS Update 存储上限

如果您超出限制或用完每月积分，任何进一步使用都将按[按使用量价格](/billing/usage-based-pricing/)收费。

### Enterprise

Enterprise 方案专为拥有大型项目且需要额外资源（如专属支持）的组织和企业设计。订阅此方案可获得：

- 可靠的生产级服务
- 用于 EAS Build 高优先级构建的每月[积分](/billing/usage-based-pricing/)
- 更多唯一用户、更高带宽以及更大的 EAS Update 存储上限

与任何其他方案相比，Enterprise 方案为 EAS Build 提供的每月积分以及 EAS Update 的带宽都要高得多。

如果您超出限制或用完每月积分，任何进一步使用都将按[按使用量价格](/billing/usage-based-pricing/)收费。

### Starter

Starter 方案面向已准备好发布真实应用的开发者。每月 19 美元，您可获得价值 45 美元的构建积分用于优先级构建。

订阅 Starter 方案可获得：

- 可靠的生产级服务
- 用于 EAS Build 高优先级构建的每月[积分](/billing/usage-based-pricing/)
- 可超出 Free 方案的 EAS Update 限制并为额外使用付费的能力

如果您超出限制或用完每月积分，任何进一步使用都将按[按使用量价格](/billing/usage-based-pricing/)收费。

## 按使用量计费

当客户超出其方案限制时，将适用按使用量计费。它使您无需担心限制或任何合同义务即可使用我们的服务。

按使用量计费按月结算，当前已启用于 EAS Build 和 EAS Update。我们会在[您的账户计费页面](https://expo.dev/settings/billing)提供您当前使用情况以及任何超额费用的估算。当您的账户达到其方案所含 EAS Build 积分的 80% 和 100% 时，Expo 还会发送[电子邮件通知](/billing/usage-based-pricing/#usage-notifications)。

## 附加项

### Enterprise Support

Enterprise Support 附加项仅面向新的 Enterprise 方案订阅者，且需视供应情况而定。主要功能包括：

- 获得我们专家提供的专业、长期支持
- 通过带有服务级别协议（SLA）的直接沟通渠道获得支持
- 专属客户经理
