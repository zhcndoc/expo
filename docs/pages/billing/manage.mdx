**Billing** 在 EAS dashboard 中提供有关你账户当前订阅计划和每月使用情况的信息。它还允许你管理计划和账单详情。

本指南说明如何管理你账户的计划和账单信息。

## 管理计划

### 查看当前计划

- 从导航菜单中的 **Subscription** 下点击 [Billing](https://expo.dev/settings/billing)。
- 在 **Current Plan** 下，你可以看到你账户当前的计划。

例如，下面这个账户订阅的是 Free 计划：

  alt="EAS dashboard 中的 Billing 页面显示了 Organization 账户的当前计划。"

/>

### 升级到新计划

要升级到其他计划：

- 在 EAS dashboard 的导航菜单中点击 [Billing](https://expo.dev/settings/billing)。
- 在 **Current Plan** 下，如果你已经在付费计划中，点击 **Change Plan**。如果你使用的是 Free 计划，点击 **See plans** > **Select your account**。这会打开 **Upgrade plan** 弹窗。
- 在 **Upgrade plan** 下，你可以看到所有可用计划的列表。选择你想要升级到的计划，然后点击目标计划下方的 **Upgrade** 按钮。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了可供升级的可用计划列表。"
  src="/static/images/billing/management/billing-03.png"
  className="max-w-[640px]"
/>

- 在 **Checkout** 中，系统会要求你输入电子邮件、卡片详情和账单地址。添加这些信息后，点击 **Pay Now** 以订阅新计划。

### 降级计划

如果你使用的是 Production、Enterprise 或 Legacy 计划，你可以降级到 Starter 计划。

降级到 Starter 计划会在你当前计费周期结束后生效。

要降级，请前往 [Billing](https://expo.dev/settings/billing) 并按照以下步骤操作：

- 在 **Current Plan** 下，点击 **Change Plan**。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于降级计划的 Change Plan 按钮。"
  src="/static/images/billing/management/billing-05.png"
  className="max-w-[480px]"
/>

- 在 **Select account** 下，从下拉菜单中选择你想要降级的账户

- 在 **Upgrade plan** > **Starter plan** 下，点击 **Change**。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于降级计划的 Change Plan 按钮。"
  src="/static/images/billing/management/billing-06.png"
  className="max-w-[640px]"
/>

- 系统会显示一个确认对话框。点击 **Done**。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于降级计划的确认对话框。"
  src="/static/images/billing/management/billing-08.png"
  className="max-w-[280px]"
/>

- 在你确认将账户降级到 Starter 计划后，同样的信息也会反映在 **Billing** > **Upcoming Plan** 下。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了所选降级计划的审阅页面。"
  src="/static/images/billing/management/billing-09.png"

### 取消计划

<Tabs>

<Tab label="从 Production 或 Enterprise 计划取消">

从 Production 或 Enterprise 计划取消会在你当前计费周期结束后生效。

要取消你的计划，请在 [Billing](https://expo.dev/settings/billing) 中，在 **Cancel all subscriptions** 下点击 **Continue to Stripe**，并按照当前计划的取消流程进行操作。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于取消计划的 Cancel Plan 按钮。"

<Tab label="从 Starter 取消到 Free 计划">

要取消你的计划，请在 [Billing](https://expo.dev/settings/billing) 中，在 **Cancel all subscriptions** 下按照提示取消你的订阅。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于取消计划的 Cancel Plan 按钮。"

## 管理账单信息

你可以管理与账单相关的详细信息，例如姓名、电子邮件、地址和付款信息，或者添加税号。所有这些信息都会列在你为所订阅计划收到的 [monthly invoice](/billing/invoices-and-receipts/) 中。

### 更新账单姓名、电子邮件或地址

要更新你的账单姓名、电子邮件或地址：

- 在 [Billing](https://expo.dev/settings/billing) 中，在 **Manage billing information** 下点击 **Manage billing**。这将打开 Stripe 的门户，在那里你可以查看付款方式、账单信息、开票历史，并更新你的账单信息。然后，点击 **Update information**。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于更新账单详情的 Update billing information 按钮。"
  src="/static/images/billing/management/billing-13.png"
  className="max-w-[640px]"
/>

- 输入你的新姓名、电子邮件或地址来更新账单详情，然后点击 **Save**。

<ContentSpotlight
  alt="在 Billing information 中，输入新的姓名、电子邮件或地址并点击 Save 以更新账单详情。"
  src="/static/images/billing/management/billing-14.png"
  className="max-w-[200px]"
/>

### 税号

要添加或更新你的账单税号：

- 在 [Billing](https://expo.dev/settings/billing) 中，在 **Manage billing information** 下点击 **Manage billing**。这将打开 Stripe 的门户，在那里你可以查看并更新你的账单信息。然后，点击 **Update information**。

<ContentSpotlight
  alt="EAS dashboard 中的 Billing 页面显示了用于更新账单详情的 Update billing information 按钮。"
  src="/static/images/billing/management/billing-13.png"
  className="max-w-[640px]"
/>

- 在 **Tax ID** 下，选择 ID 类型，输入有效的税号，然后点击 **Save**。

<ContentSpotlight
  alt="在 Billing information 中，输入新的税号并点击 Save 以更新账单详情。"
  src="/static/images/billing/management/billing-15.png"
  className="max-w-[200px]"
/>

### 付款方式

要添加新的付款方式信息：

- 在 [Billing](https://expo.dev/settings/billing) 中，在 **Manage billing information** 下点击 **Manage billing**。这将打开 Stripe 的门户，在那里你可以查看并更新你的账单信息。
- 在 **Payment method** 下，点击 **Add payment method** 以添加新的付款方式。

<ContentSpotlight
  alt="在 Billing information 中，你可以添加新的付款方式。"
  src="/static/images/billing/management/payment-01.png"
  className="max-w-[400px]"
/>

- 输入新的付款方式详情，然后点击 **Add**。

  alt="添加新的付款方式时，输入新的付款方式详情。"
