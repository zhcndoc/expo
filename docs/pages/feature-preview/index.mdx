### 功能预览是什么意思？

我们决定将一些新功能以预览版的形式发布，以便从您那里获得反馈。这并不意味着这些功能处于“beta”状态。作为开发者，我们对依赖任何标记为“beta”的内容都持怀疑态度，因为我们知道这个标签有时会被用于一些开发早期、尚未完成，或者可能不会获得长期支持的项目。Expo 为您提供经过测试并被许多团队采用的解决方案，帮助您处理日常工作。这些功能之所以以预览版发布，是为了倾听您的反馈并将其改进。我们将这段时间视为更好地了解开发者对我们服务的期望，并据此调整它们以适应您的需求。在进行这些改进的同时，我们预计这些包中出现破坏性变更的频率会比 Expo SDK 更高。
