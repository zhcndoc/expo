本文档的这一部分提供了我们在 Expo 正在开发的实验性新功能预览。它没有从主文档中链接出来，但也并没有特别隐藏。

> **info** `/preview` 下的页面不会被搜索引擎索引。

在探索预览功能时，我们只希望你牢记以下几点：

1. **不要撰写博客文章、YouTube 视频或其他涉及预览功能的内容。** 这里记录的一切都很可能发生变化，内容很快就会过时。
2. **你可能不应该在生产环境中依赖预览功能**。除非你已经准备好应对这个快速演进的产品可能带来的不稳定性。
3. **可免费预览的功能在正式发布后可能会变成付费服务**。不要基于这些功能将你的业务建立在“免费”这一前提上。
4. **请向我们提供你在使用实验性功能时的详细反馈**。从文档到 API 接口，再到 CLI 命令，我们都希望听到你的反馈，因为在预览阶段之外，几乎没有比现在更容易修改它的时候了。关于这一点的更多信息，请参见 [支持与反馈](#support-and-feedback)。

## 支持与反馈

功能预览的目的是在我们将产品推向更广泛的受众之前收集反馈，以打磨和改进产品。我们希望你来试用这些功能，并告诉我们使用情况如何。请在你的最大型应用、较小的应用、新应用、旧应用，以及你有耐心和好奇心去尝试的各种场景中测试它们。

请将你的所有支持问题和反馈发送到某个合作伙伴 Slack 频道（如果你有的话，你应该知道是哪一个），或者使用 **分享你的反馈**，也可以通过 [preview@expo.dev](mailto:preview@expo.dev) 给我们发送电子邮件。

谢谢！现在请查看导航菜单以探索这些预览功能。
