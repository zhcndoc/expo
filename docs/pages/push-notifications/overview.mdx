Expo 通过处理与 Firebase Cloud Messaging (FCM) 或 Apple Push Notification Service (APNs) 通信中涉及的大部分复杂性，简化了推送通知的实现。这使你可以以相同的方式处理 Android 和 iOS 通知，并在前端和后端都节省时间。

请观看下面的视频或点击链接，了解如何在应用中设置推送通知、发送推送通知以及处理接收到的通知。
