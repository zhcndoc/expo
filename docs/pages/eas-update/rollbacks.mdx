---
title: 回滚
description: 将分支回滚到之前的更新或内置更新。
---

import { Terminal } from '~/ui/components/Snippet';

EAS Update 支持两种回滚类型：

- 回滚到之前发布的更新。
- 回滚到构建中嵌入的更新。

## 开始回滚

要开始回滚，请运行以下命令：

<Terminal cmd={['$ eas update:rollback']} />

在终端中，将有一个交互式向导帮助你选择回滚类型并执行回滚。

## 回滚到之前发布的更新

上述命令会重新发布之前发布的更新，从功能上将客户端回滚到该更新。

## 回滚到构建中嵌入的更新

上述命令会指示客户端运行构建中嵌入的更新。

## 在回滚后再次发布

在回滚后再次发布时，所有客户端都将收到新更新。
