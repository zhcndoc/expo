Expo Go 游乐场使学生和学习者能够快速尝试构建原生 Android 和 iOS 应用。它支持一份经过筛选的第三方库列表，这些库由社区驱动，提供常见应用功能的 API，并且会在每个 Expo SDK 版本发布时进行测试。

本节中列出的库与其他任何第三方库之间的区别在于，前者在 Expo Go 环境中是内置支持的。你可以在开发构建中使用你选择的任何库。
