`expo-linking` 为你的应用提供了与其他已安装应用通过深度链接进行交互的工具。它还提供了用于构建和解析进入你应用的深度链接的辅助方法。这个库是 React Native [`Linking`](https://reactnative.dev/docs/linking) 的扩展。

有关如何使用 `expo-linking` 的更全面说明，请参阅 [链接到其他应用](/linking/into-other-apps/)。
