`expo-av` 中的 `Video` 组件会将视频以内嵌方式显示在应用的其他 UI 元素中。

Video 和 Audio 的很多 API 是通用的，这些内容记录在 [AV 文档](av.mdx) 中。本页面涵盖仅针对视频的属性和 API。我们建议你先快速浏览本文档，以便让基础视频功能正常工作，然后再前往 [AV 文档](av.mdx) 了解更高级的功能。视频的音频体验（例如是否打断另一个应用中正在播放的音乐，或在手机静音模式下是否播放声音）可以使用 [Audio API](audio.mdx) 自定义。

## 用法

下面是一个带有播放/暂停按钮的视频简单示例。

<SnackInline label='Video' dependencies={['expo-av', 'expo-asset']}>

```jsx
import { useState, useRef } from 'react';
import { View, StyleSheet, Button } from 'react-native';
import { Video, ResizeMode } from 'expo-av';

export default function App() {
  const video = useRef(null);
  const [status, setStatus] = useState({});
  return (
    <View style={styles.container}>
      <Video
        ref={video}
        style={styles.video}
        source={{
          uri: 'https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4',
        }}
        useNativeControls
        resizeMode={ResizeMode.CONTAIN}
        isLooping
        onPlaybackStatusUpdate={status => setStatus(() => status)}
      />
      <View style={styles.buttons}>
        <Button
          title={status.isPlaying ? '暂停' : '播放'}
          onPress={() =>
            status.isPlaying ? video.current.pauseAsync() : video.current.playAsync()
          }
        />
      </View>
    </View>
  );
}

/* @hide const styles = StyleSheet.create({ ... }); */
const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    backgroundColor: '#ecf0f1',
  },
  video: {
    alignSelf: 'center',
    width: 320,
    height: 200,
  },
  buttons: {
    flexDirection: 'row',
    justifyContent: 'center',
    alignItems: 'center',
  },
});
/* @end */
```

</SnackInline>

如需查看更多高级示例，请查看 [Playlist 示例](https://github.com/expo/playlist-example/blob/master/App.js) 以及 [自定义 `VideoPlayer` 控件组件](https://github.com/ihmpavel/expo-video-player/blob/master/lib/index.tsx)，它封装了 `<Video>`、添加了自定义控件，并大量使用了 `<Video>` API。`VideoPlayer` 控件用于 [这个应用](https://github.com/expo/harvard-cs50-app) 中。
