`expo-intent-launcher` 提供了一种启动 Android intent 的方式。例如，你可以使用此 API 打开特定的设置页面。

## 安装

<APIInstallSection />

## 用法

```ts
import { startActivityAsync, ActivityAction } from 'expo-intent-launcher';

// 打开位置设置
startActivityAsync(ActivityAction.LOCATION_SOURCE_SETTINGS);
```
