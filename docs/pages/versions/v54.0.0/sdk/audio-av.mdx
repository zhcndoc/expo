`expo-av` 中的 `Audio` 允许你在应用中实现音频播放和录制。

请注意，如果耳机/蓝牙音频设备断开连接，音频会自动停止。

有关媒体播放 API 的示例，请参见 [playlist example app](https://github.com/expo/playlist-example)；有关录制 API 的示例，请参见 [recording example app](https://github.com/expo/audio-recording-example)。

### 播放声音

  label='Playing sounds'
  dependencies={['expo-av', 'expo-asset']}

    'assets/Hello.mp3': 'https://snack-code-uploads.s3.us-west-1.amazonaws.com/~asset/c9c43b458d6daa9771a7287cae9f5b47'

```jsx
import { useEffect, useState } from 'react';
import { View, StyleSheet, Button } from 'react-native';
import { Audio } from 'expo-av';

export default function App() {
  const [sound, setSound] = useState();

  async function playSound() {
    console.log('加载声音');
    /* @info */ const { sound } = await Audio.Sound.createAsync(
      /* @end */ require('./assets/Hello.mp3')
    );
    setSound(sound);

    console.log('播放声音');
    await /* @info */ sound.playAsync(); /* @end */
  }

  useEffect(() => {
    return sound
      ? () => {
          console.log('卸载声音');
          /* @info 使用后请务必卸载 Sound，以防止内存泄漏。*/ sound.unloadAsync(); /* @end */
        }
      : undefined;
  }, [sound]);

  return (
    <View style={styles.container}>
      <Button title="播放声音" onPress={playSound} />
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    backgroundColor: '#ecf0f1',
    padding: 10,
  },
});
```

### 录制声音

<SnackInline label='录制声音' dependencies={['expo-av', 'expo-asset']}>

```jsx
import { useState } from 'react';
import { View, StyleSheet, Button } from 'react-native';
import { Audio } from 'expo-av';

export default function App() {
  const [recording, setRecording] = useState();
  const [permissionResponse, requestPermission] = Audio.usePermissions();

  async function startRecording() {
    try {
      /* @info */ if (permissionResponse.status !== 'granted') {
        console.log('请求权限..');
        await requestPermission();
      }
      await Audio.setAudioModeAsync({
        allowsRecordingIOS: true,
        playsInSilentModeIOS: true,
      }); /* @end */

      console.log('开始录制..');
      /* @info */ const { recording } = await Audio.Recording.createAsync(
        /* @end */ Audio.RecordingOptionsPresets.HIGH_QUALITY
      );
      setRecording(recording);
      console.log('录制已开始');
    } catch (err) {
      console.error('开始录制失败', err);
    }
  }

  async function stopRecording() {
    console.log('停止录制..');
    setRecording(undefined);
    /* @info */ await recording.stopAndUnloadAsync(); /* @end */
    /* @info 当允许录制时，iOS 可能会将音频播放路由到手机听筒，因此完成后请将其禁用。 */ await Audio.setAudioModeAsync(
      {
        allowsRecordingIOS: false,
      }
    ); /* @end */
    /* @info */ const uri = recording.getURI(); /* @end */
    console.log('录制已停止并存储在', uri);
  }

  return (
    <View style={styles.container}>
      <Button
        title={recording ? '停止录制' : '开始录制'}
        onPress={recording ? stopRecording : startRecording}
      />
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    backgroundColor: '#ecf0f1',
    padding: 10,
  },
});
```

### 在后台播放或录制音频&ensp;<PlatformTags platforms={['ios']} />

在 iOS 上，后台音频播放和录制仅适用于独立应用，并且需要一些额外配置。
在 iOS 上，每个后台功能都需要在 **Info.plist** 文件中的 `UIBackgroundModes` 数组里设置一个特殊键。
在独立应用中，此数组默认为空，因此要使用后台功能，你需要在 **app.json** 配置中添加相应的键。

以下是启用后台音频播放的 **app.json** 示例：

```json
{
  "expo": {
    ...
    "ios": {
      ...
      "infoPlist": {
        ...
        "UIBackgroundModes": [
          "audio"
        ]
      }
    }
  }
}
```

### 关于 web 使用的说明

- Chrome 上的 MediaRecorder 问题会生成缺少时长元数据的 WebM 文件。[查看开放的 Chromium 问题](https://bugs.chromium.org/p/chromium/issues/detail?id=642012)。
- MediaRecorder 的编码选项和其他配置在不同浏览器之间并不一致，在你的应用中使用诸如 [kbumsik/opus-media-recorder](https://github.com/kbumsik/opus-media-recorder) 或 [ai/audio-recorder-polyfill](https://github.com/ai/audio-recorder-polyfill) 这样的 Polyfill 会改善你的体验。传递给 `prepareToRecordAsync` 的任何选项都会直接传递给 MediaRecorder API，因此也会传递给该 polyfill。
- Web 浏览器要求网站以安全方式提供服务，才能监听麦克风。更多详情请参见 [MediaDevices `getUserMedia()` security](https://developer.mozilla.org/en-US/docs/Web/API/MediaDevices/getUserMedia#security)。
