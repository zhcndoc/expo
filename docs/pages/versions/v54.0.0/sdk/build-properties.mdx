`expo-build-properties` 是一个 [config plugin](/config-plugins/introduction/)，用于在 [Prebuild](/workflow/continuous-native-generation) 期间配置你的 **android/gradle.properties** 和 **ios/Podfile.properties.json** 目录中的原生构建属性。

> **info** 此配置插件用于配置 [Prebuild command](/workflow/continuous-native-generation/#usage) 生成原生 **android** 和 **ios** 目录的方式
> 因此不能用于不运行 `npx expo prebuild` 的项目（裸工作流项目）。

## 用法

<ConfigPluginExample>

<Tabs>

<Tab label="app.json">

```json app.json|collapseHeight=450
{
  "expo": {
    "plugins": [
      [
        "expo-build-properties",
        {
          "android": {
            "compileSdkVersion": 35,
            "targetSdkVersion": 35,
            "buildToolsVersion": "35.0.0"
          },
          "ios": {
            "deploymentTarget": "15.1"
          }
        }
      ]
    ]
  }
}
```

</Tab>

<Tab label="app.config.js">

```js app.config.js|collapseHeight=450
export default {
  expo: {
    plugins: [
      [
        'expo-build-properties',
        {
          android: {
            compileSdkVersion: 35,
            targetSdkVersion: 35,
            buildToolsVersion: '35.0.0',
          },
          ios: {
            deploymentTarget: '15.1',
          },
        },
      ],
    ],
  },
};
```

</Tab>

</Tabs>

</ConfigPluginExample>
