一个 Expo Router API，提供用于在路由之间导航的组件，包括 link、redirect、preview 和 zoom 过渡效果。

> 请参阅 [Expo Router](./index) 参考文档以了解安装和配置。

## 用法

```tsx
import { Link } from 'expo-router';

export default function Page() {
  return <Link href="/about">关于</Link>;
}
```
