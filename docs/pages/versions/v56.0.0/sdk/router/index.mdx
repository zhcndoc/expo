## 在 app 配置中进行配置

如果你正在使用 [默认](/more/create-expo/#--template) 模板创建新项目，那么 `expo-router` 的 [配置插件](/config-plugins/introduction/) 已经在你的 app 配置中设置好了。

<ConfigPluginExample>

```json app.json
{
  "expo": {
    "plugins": ["expo-router"]
  }
}
```

</ConfigPluginExample>
