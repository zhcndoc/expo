> **重要** `ExperimentalStack` 是一个可在 **Expo SDK 56** 及更高版本中使用的 [alpha](/more/release-statuses/#alpha) API。它仅用于测试 &mdash; 在准备好用于生产环境之前，API 和功能集都可能发生变化。

`ExperimentalStack` 是 [`Stack`](./stack) 的同级组件，由新的 `react-native-screens/experimental` 栈提供支持。它需要按导航器逐个启用：在你想迁移的特定布局中，将 `<Stack />` 替换为 `<ExperimentalStack />`，并在其他所有地方继续使用 `<Stack />`。

我们提前分享它，是为了让你可以在应用中试用并告诉我们还缺少什么。当前支持的选项范围是刻意限制的，并且会随着时间推移逐步扩展。

> 有关适用于原生和 Web 应用的基于文件的路由库的更多信息，请参阅 [Expo Router](./index) 参考文档。

## 支持的功能

支持的屏幕选项：

- `title`
- `headerShown`
- `headerTransparent`
- `headerBackVisible`

在 Android 上，`ExperimentalStack` 支持 [预测性返回手势](https://developer.android.com/guide/navigation/custom-back/predictive-back-gesture)。你仍然需要在你的 [app config](/workflow/configuration/) 中将 [`android.predictiveBackGestureEnabled`](/versions/latest/config/app/#predictivebackgestureenabled) 设置为 `true`，才能为应用启用它。

## 平台支持

`ExperimentalStack` 仅支持原生平台。在 Web 上，它会回退到标准的 `Stack`，因此同一布局可以在各平台上工作，而无需条件代码。

## 基本用法

```tsx app/_layout.tsx
import { ExperimentalStack as Stack } from 'expo-router';

export default function Layout() {
  return (
    <Stack screenOptions={{ headerShown: true }}>
      <Stack.Screen name="index" options={{ title: 'Home' }} />
      <Stack.Screen name="details" options={{ title: 'Details' }} />
    </Stack>
  );
}
```

你可以像使用 `Stack` 一样组合使用 `ExperimentalStack.Screen` 和 `ExperimentalStack.Protected`。

## 已知限制

<Collapsible summary="屏幕选项有限">

`ExperimentalStack` 仅支持 `title`、`headerShown`、`headerTransparent` 和 `headerBackVisible`。传递任何其他选项（例如 `headerLeft`、`headerRight`、`headerTitle`、`headerStyle`、`headerTintColor`、动画覆盖项、状态栏选项）都会在开发环境中记录警告，并且不会生效。对于需要这些选项的屏幕，请继续使用 `<Stack />`。

</Collapsible>

<Collapsible summary="尚不支持展示模式">

`ExperimentalStack` 目前还不支持 `presentation: 'modal'` 或 `transparentModal`。屏幕始终会压入栈中。

</Collapsible>

<Collapsible summary="尚不支持 sheet">

`ExperimentalStack` 目前还不支持 `formSheet` 或相关的 sheet 尺寸/detent 选项。

</Collapsible>

<Collapsible summary="尚不支持自定义头部">

`ExperimentalStack` 目前还不支持自定义头部组件或头部着色/样式设置。只有上面列出的四个头部选项会生效。

</Collapsible>

<Collapsible summary="尚不支持动画或状态栏自定义">

`ExperimentalStack` 目前还不会遵循按屏幕覆盖的动画设置（`animation`、`animationDuration`）或状态栏选项。

</Collapsible>

<Collapsible summary="在 Android 上不能与 Stack 混用">

在 Android 上，`ExperimentalStack` 和标准的 `Stack` 不能共存于同一个应用中 &mdash; 请为你的原生栈选择一种导航器类型。我们希望在未来的版本中取消这一限制，这样你就可以一次迁移一个导航器。

</Collapsible>

<Collapsible summary="Web 会回退到标准 Stack">

在 Web 上，`<ExperimentalStack />` 会渲染来自 `expo-router` 的标准 `Stack`。仅适用于原生平台的选项在 Web 上不会生效。

</Collapsible>
