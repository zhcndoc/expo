一个提供 Stack 导航器、工具栏和屏幕组件的 Expo Router API。

> 有关安装和配置，请参阅 [Expo Router](./index) 参考文档。

## 用法

```tsx
import { Stack } from 'expo-router';

export default function Layout() {
  return <Stack />;
}
```
