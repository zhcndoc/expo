一个水平布局容器，从起始到结束排列其子元素。在 Android 上委托给 Jetpack Compose 的 [`Row`](../jetpack-compose/row)，在 iOS 上委托给 SwiftUI 的 [`HStack`](../swift-ui/hstack)，在 Web 上委托给 flex `View`。

## 用法

### 基本行

```tsx RowExample.tsx
import { Host, Row, Text } from '@expo/ui';

export default function RowExample() {
  return (
    <Host matchContents>
      <Row spacing={8}>
        <Text>One</Text>
        <Text>Two</Text>
        <Text>Three</Text>
      </Row>
    </Host>
  );
}
```

### 对齐

使用 [`alignment`](#alignment) 在交叉（垂直）轴上定位子元素。

```tsx RowAlignmentExample.tsx
import { Host, Row, Text } from '@expo/ui';

export default function RowAlignmentExample() {
  return (
    <Host style={{ flex: 1 }}>
      <Row spacing={8} alignment="center">
        <Text>居中</Text>
        <Text>居中</Text>
      </Row>
    </Host>
  );
}
```

### 使用 Spacer 拉开内容

将 `Row` 与可伸缩的 [`Spacer`](spacer) 配对，以将其子元素推到相对两端。

```tsx RowSpacerExample.tsx
import { Host, Row, Text, Spacer } from '@expo/ui';

export default function RowSpacerExample() {
  return (
    <Host style={{ flex: 1 }}>
      <Row>
        <Text>前导</Text>
        <Spacer flexible />
        <Text>尾随</Text>
      </Row>
    </Host>
  );
}
```
