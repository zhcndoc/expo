一个布局间隔组件，用于在 [`Row`](row) 或 [`Column`](column) 内的兄弟元素之间创建空白空间。使用 [`size`](#size) 设置固定间隙，或使用 [`flexible`](#flexible) 填充剩余的主轴空间。

## 用法

### 固定大小间隔

```tsx FixedSpacerExample.tsx
import { Host, Column, Text, Spacer } from '@expo/ui';

export default function FixedSpacerExample() {
  return (
    <Host matchContents>
      <Column>
        <Text>顶部</Text>
        <Spacer size={32} />
        <Text>底部</Text>
      </Column>
    </Host>
  );
}
```

### 可伸缩间隔

可伸缩间隔会沿其父级的主轴填充剩余空间，将周围内容推到两端。

```tsx FlexibleSpacerExample.tsx
import { Host, Row, Text, Spacer } from '@expo/ui';

export default function FlexibleSpacerExample() {
  return (
    <Host style={{ flex: 1 }}>
      <Row>
        <Text>前导</Text>
        <Spacer flexible />
        <Text>尾随</Text>
      </Row>
    </Host>
  );
}
```
