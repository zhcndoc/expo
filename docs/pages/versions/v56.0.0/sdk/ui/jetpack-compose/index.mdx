## 用法

使用 `@expo/ui/jetpack-compose` 中的组件需要将其包装在一个 [`Host`](./host) 组件中。`Host` 是 Jetpack Compose 视图的容器。

```tsx
import { Host, Button } from '@expo/ui/jetpack-compose';

export function SaveButton() {
  return (
    <Host matchContents>
      <Button onClick={() => alert('已保存！')}>保存更改</Button>
    </Host>
  );
}
```
