`Host` 组件是 React Native 和 Jetpack Compose 之间的桥梁。来自 `@expo/ui/jetpack-compose` 的每个 Jetpack Compose 组件都必须包装在 `Host` 中才能正确渲染。

## 用法

### 匹配内容

使用 `matchContents` 属性让 `Host` 的尺寸自动适配内容。你可以传入布尔值或对象，以独立控制垂直和水平方向的尺寸。

```tsx MatchContents.tsx
import { Host, Button } from '@expo/ui/jetpack-compose';

export default function MatchContents() {
  return (
    <Host matchContents>
      <Button onClick={() => console.log('Pressed')}>根据内容调整大小</Button>
    </Host>
  );
}
```

> **注意：** 不要在与可滚动子组件（`LazyRow`、`LazyColumn`、`Carousel`，或任何使用 `Modifier.horizontalScroll`/`verticalScroll` 的组件）相同的轴上使用 `matchContents`。可滚动组件在其滚动轴上需要有限的最大约束，而 `matchContents` 会传递一个无限约束。

下面的示例会崩溃：

```tsx MatchContentsCrash.tsx
import { Host, LazyRow, Text } from '@expo/ui/jetpack-compose';

export default function MatchContentsCrash() {
  return (
    <Host matchContents>
      <LazyRow>
        {Array.from({ length: 5 }).map((_, i) => (
          <Text key={i}>项目 {i}</Text>
        ))}
      </LazyRow>
    </Host>
  );
}
```

要么在滚动轴上去掉 `matchContents`，要么通过 `style` 为 `Host` 在该轴上提供一个有限大小：

```tsx MatchContentsFix.tsx
import { Host, LazyRow, Text } from '@expo/ui/jetpack-compose';

export default function MatchContentsFix() {
  return (
    <Host matchContents={{ vertical: true }} style={{ width: '100%' }}>
      <LazyRow>
        {Array.from({ length: 5 }).map((_, i) => (
          <Text key={i}>项目 {i}</Text>
        ))}
      </LazyRow>
    </Host>
  );
}
```

### 配合样式

将标准的 React Native 样式应用到 `Host` 包装器上。

```tsx HostWithStyle.tsx
import { Host, Button } from '@expo/ui/jetpack-compose';

export default function HostWithStyle() {
  return (
    <Host style={{ padding: 16, backgroundColor: '#f0f0f0', borderRadius: 8 }}>
      <Button onClick={() => console.log('Pressed')}>带样式的 host</Button>
    </Host>
  );
}
```
