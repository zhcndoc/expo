## 用法

使用 `@expo/ui/swift-ui` 中的组件时，需要将其包装在 [`Host`](./host) 组件中。`Host` 是 SwiftUI 视图的容器。

```tsx
import { Host, Button } from '@expo/ui/swift-ui';

export function SaveButton() {
  return (
    <Host style={{ flex: 1 }}>
      <Button label="保存更改" />
    </Host>
  );
}
```
