`expo-network` 提供有关设备网络的有用信息，例如其 IP 地址、MAC 地址和飞行模式状态。

## 配置

在 Android 上，此模块需要权限以访问网络和 Wi-Fi 状态。权限 `ACCESS_NETWORK_STATE` 和 `ACCESS_WIFI_STATE` 会自动添加。

import * as Network from 'expo-network';

## 错误代码

| 代码                                    | 描述                                                                                                                                                                                |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ERR_NETWORK_IP_ADDRESS                  | 在 Android 上，尝试在 `getIpAddressAsync` 中访问 `WifiManager` 时，可能会遇到未知的 Wi-Fi 主机。在 iOS 上，无法检索任何网络接口。                               |
| ERR_NETWORK_UNDEFINED_INTERFACE         | 在 `getMacAddressAsync` 中作为参数传入了未定义的 `interfaceName`。                                                                                                            |
| ERR_NETWORK_SOCKET_EXCEPTION            | 在 `getMacAddressAsync` 中创建或访问套接字时遇到错误。                                                                                                      |
| ERR_NETWORK_INVALID_PERMISSION_INTERNET | 在 `getMacAddressAsync` 中，[`android.permission.ACCESS_WIFI_STATE`](https://developer.android.com/reference/android/Manifest.permission#ACCESS_WIFI_STATE) 的权限无效。 |
| ERR_NETWORK_NO_ACCESS_NETWORKINFO       | 无法访问网络信息                                                                                                                                                       |
