`expo-linking` 提供了实用工具，帮助你的应用使用深度链接与其他已安装的应用交互。它还提供了用于构建和解析指向你应用的深度链接的辅助方法。这个库是 React Native [`Linking`](https://reactnative.dev/docs/linking) 的扩展。

如需更全面地了解如何使用 `expo-linking`，请参阅 [链接到其他应用](/linking/into-other-apps/)。
