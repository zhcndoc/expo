`@react-native-masked-view/masked-view` 提供了一个蒙版视图，它只显示与其 mask 元素中渲染的视图重叠的像素。

> **警告** 你的项目中任意时刻只能安装 `@react-native-community/masked-view`（已弃用）或 `@react-native-masked-view/masked-view` 其中之一。React Navigation v6 及更高版本需要 `@react-native-masked-view/masked-view`，因此如果你使用的是最新版本的 React Navigation，应改用该包。

> **重要** 此库对 Android 的支持仍处于实验阶段，你可能会遇到不同平台之间行为不一致的情况。请将你遇到的问题报告到 [`react-native-masked-view` GitHub 仓库](https://github.com/react-native-masked-view/masked-view)。
