它有点像一个花哨的单选按钮，或者用 Apple 的话来说：“一种由多个分段组成的水平控件，每个分段都作为一个独立按钮发挥作用”([来源](https://developer.apple.com/documentation/uikit/uisegmentedcontrol))。这个组件在 iOS 上会渲染为一个 [`UISegmentedControl`](https://developer.apple.com/documentation/uikit/uisegmentedcontrol)，并在 Android 和 web 上忠实复刻该控件（因为这些平台的标准库中没有对应的等价控件）。
