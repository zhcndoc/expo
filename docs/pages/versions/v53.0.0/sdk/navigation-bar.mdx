`expo-navigation-bar` 使您能够修改和观察 Android 设备上的原生导航栏。由于某些 Android 平台限制，此 API 的部分功能与 `expo-status-bar` API 重叠。

属性以样式属性命名；例如 visibility、position、backgroundColor、borderColor 等。

当 Android 设备启用了“手势导航”时，此包中的 API 不会产生任何影响。目前还没有原生 Android API 可以检测是否启用了“手势导航”。
