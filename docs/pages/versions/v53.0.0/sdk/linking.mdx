`expo-linking` 提供了用于让你的应用通过深度链接与其他已安装应用交互的工具。它还提供了辅助方法，用于构造和解析指向你应用的深度链接。这个库是 React Native [`Linking`](https://reactnative.dev/docs/linking) 的扩展。

如需更全面地了解如何使用 `expo-linking`，请参阅[链接到其他应用](/linking/into-other-apps/)。
