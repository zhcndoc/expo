`expo-router` 是一个适用于 React Native 和 web 应用的路由库。它通过基于文件的路由系统实现导航管理，并提供原生导航组件，构建于 [React Navigation](https://reactnavigation.org/) 之上。

## 在 app 配置中的配置

如果你使用 [默认](/more/create-expo/#--template) 模板创建新项目，`expo-router` 的 [配置插件](/config-plugins/introduction/) 已经在你的 app 配置中完成设置。

<ConfigPluginExample>

```json app.json
{
  "expo": {
    "plugins": ["expo-router"]
  }
}
```
