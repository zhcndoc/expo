`@expo/ui` 是一组原生输入组件，可让你使用 Jetpack Compose 和 SwiftUI 构建完全原生的界面。它旨在提供典型应用所需的常用功能和组件。

## 可用平台

组件适用于以下平台：

- **[Jetpack Compose](jetpack-compose)**: 使用 Jetpack Compose 组件构建原生 Android 界面
- **[SwiftUI](swift-ui)**: 使用 SwiftUI 组件构建原生 iOS 界面
- **[Universal](universal)**: 可在 Android、iOS 和 web 上运行的跨平台组件，来自单一代码源
