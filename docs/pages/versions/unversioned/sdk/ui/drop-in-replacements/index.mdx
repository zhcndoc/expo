以下组件提供与流行的 React Native 社区库 API 兼容的替代方案，由 `@expo/ui` 原生组件（Android 上的 Jetpack Compose 和 iOS 上的 SwiftUI）提供支持。

- **[DateTimePicker](datetimepicker)**: 与 `@react-native-community/datetimepicker` 兼容
- **[BottomSheet](bottomsheet)**: 与 `@gorhom/bottom-sheet` 兼容
- **[Picker](picker)**: 与 `@react-native-picker/picker` 兼容
- **[SegmentedControl](segmentedcontrol)**: 与 `@react-native-segmented-control/segmented-control` 兼容
- **[Slider](slider)**: 与 `@react-native-community/slider` 兼容
- **[MaskedView](maskedview)**: 与 `@react-native-masked-view/masked-view` 兼容
