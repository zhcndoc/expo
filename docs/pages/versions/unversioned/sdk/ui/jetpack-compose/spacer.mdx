Expo UI Spacer 与官方 Jetpack Compose [Spacer](https://developer.android.com/reference/kotlin/androidx/compose/foundation/layout/package-summary#Spacer) API 保持一致，用于在布局中的元素之间添加可伸缩或固定大小的间距。

## 用法

### 带权重的 Spacer

使用 `weight()` 修饰符让 spacer 在 `Row` 或 `Column` 中按比例填充可用空间。

```tsx SpacerWeightExample.tsx
import { Host, Row, Spacer, Text } from '@expo/ui/jetpack-compose';
import { fillMaxWidth, weight } from '@expo/ui/jetpack-compose/modifiers';

export default function SpacerWeightExample() {
  return (
    <Host matchContents>
      <Row modifiers={[fillMaxWidth()]}>
        <Text>左侧</Text>
        <Spacer modifiers={[weight(1)]} />
        <Text>右侧</Text>
      </Row>
    </Host>
  );
}
```

### 固定大小的 Spacer

使用 `height` 或 `width` 修饰符创建一个具有固定尺寸的 spacer。

```tsx SpacerFixedSizeExample.tsx
import { Host, Column, Spacer, Text } from '@expo/ui/jetpack-compose';
import { height } from '@expo/ui/jetpack-compose/modifiers';

export default function SpacerFixedSizeExample() {
  return (
    <Host matchContents>
      <Column>
        <Text>上方</Text>
        <Spacer modifiers={[height(24)]} />
        <Text>下方（24dp 间距）</Text>
      </Column>
    </Host>
  );
}
```
