`Host` 组件是 React Native 和 Jetpack Compose 之间的桥梁。来自 `@expo/ui/jetpack-compose` 的每个 Jetpack Compose 组件都必须包装在 `Host` 中才能正确渲染。

## 用法

### 匹配内容

使用 `matchContents` 属性让 `Host` 的尺寸自动适配内容。你可以传入布尔值或对象，以分别控制垂直和水平方向的尺寸。

```tsx MatchContents.tsx
import { Host, Button } from '@expo/ui/jetpack-compose';

export default function MatchContents() {
  return (
    <Host matchContents>
      <Button onClick={() => console.log('Pressed')}>按内容自适应尺寸</Button>
    </Host>
  );
}
```

### 使用样式

将标准 React Native 样式应用到 `Host` 包装器上。

```tsx HostWithStyle.tsx
import { Host, Button } from '@expo/ui/jetpack-compose';

export default function HostWithStyle() {
  return (
    <Host style={{ padding: 16, backgroundColor: '#f0f0f0', borderRadius: 8 }}>
      <Button onClick={() => console.log('Pressed')}>带样式的 Host</Button>
    </Host>
  );
}
```
