一种垂直布局容器，将其子元素从上到下排列。在 iOS 上委托给 SwiftUI 的 [`VStack`](../swift-ui/vstack)，在 Android 上委托给 Jetpack Compose 的 [`Column`](../jetpack-compose/column)，在 Web 上委托给 flex `View`。

## 用法

### 基本列

```tsx ColumnExample.tsx
import { Host, Column, Text } from '@expo/ui';

export default function ColumnExample() {
  return (
    <Host matchContents>
      <Column spacing={8}>
        <Text>第一</Text>
        <Text>第二</Text>
        <Text>第三</Text>
      </Column>
    </Host>
  );
}
```

### 对齐

使用 [`alignment`](#alignment) 沿交叉（水平）轴定位子元素。

```tsx ColumnAlignmentExample.tsx
import { Host, Column, Text } from '@expo/ui';

export default function ColumnAlignmentExample() {
  return (
    <Host style={{ flex: 1 }}>
      <Column spacing={8} alignment="center">
        <Text>居中</Text>
        <Text>居中</Text>
      </Column>
    </Host>
  );
}
```
