`expo-build-properties` 是一个 [配置插件](/config-plugins/introduction/)，用于在 [预构建](/workflow/continuous-native-generation) 期间配置 **android/gradle.properties** 和 **ios/Podfile.properties.json** 目录的原生构建属性。

> **info** 此配置插件用于配置 [预构建命令](/workflow/continuous-native-generation/#usage) 如何生成原生的 **android** 和 **ios** 目录，
> 因此不能用于不运行 `npx expo prebuild` 的项目（裸项目）。

## 用法

<ConfigPluginExample>

<Tabs>

<Tab label="app.json">

```json app.json|collapseHeight=450
{
  "expo": {
    "plugins": [
      [
        "expo-build-properties",
        {
          "android": {
            "compileSdkVersion": {{compileSdkVersion}},
            "targetSdkVersion": {{targetSdkVersion}},
            "buildToolsVersion": "{{buildToolsVersion}}"
          },
          "ios": {
            "deploymentTarget": "{{iosDeploymentTarget}}"
          }
        }
      ]
    ]
  }
}
```

</Tab>

<Tab label="app.config.js">

```js app.config.js|collapseHeight=450
export default {
  expo: {
    plugins: [
      [
        'expo-build-properties',
        {
          android: {
            compileSdkVersion: {{compileSdkVersion}},
            targetSdkVersion: {{targetSdkVersion}},
            buildToolsVersion: '{{buildToolsVersion}}',
          },
          ios: {
            deploymentTarget: '{{iosDeploymentTarget}}',
          },
        },
      ],
    ],
  },
};
```

</Tab>

</Tabs>

</ConfigPluginExample>
