`expo-linking` 提供了用于让你的应用通过深度链接与其他已安装应用交互的实用工具。它还提供了帮助方法，用于构建和解析指向你应用的深度链接。这个库是 React Native [`Linking`](https://reactnative.dev/docs/linking) 的扩展。

有关如何使用 `expo-linking` 的更全面说明，请参阅[链接到其他应用](/linking/into-other-apps/)。
