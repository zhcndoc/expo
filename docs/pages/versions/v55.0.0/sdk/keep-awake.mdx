`expo-keep-awake` 提供了一个 React Hook，用于防止屏幕进入休眠，并提供了一对函数以命令式地启用此行为。

## 用法

### 示例：hook

<SnackInline label='Keep Awake hook' dependencies={['expo-keep-awake']}>

```jsx
import { useKeepAwake } from 'expo-keep-awake';
import React from 'react';
import { Text, View } from 'react-native';

export default function KeepAwakeExample() {
  /* @info 只要这个组件处于挂载状态，屏幕就不会因为空闲而关闭。 */
  useKeepAwake();
  /* @end */
  return (
    <View style={{ flex: 1, alignItems: 'center', justifyContent: 'center' }}>
      <Text>This screen will never sleep!</Text>
    </View>
  );
}
```

</SnackInline>

### 示例：函数

<SnackInline label='Keep Awake functions' dependencies={['expo-keep-awake']}>

```jsx
import { activateKeepAwake, deactivateKeepAwake } from 'expo-keep-awake';
import React from 'react';
import { Button, View } from 'react-native';

export default class KeepAwakeExample extends React.Component {
  render() {
    return (
      <View style={{ flex: 1, alignItems: 'center', justifyContent: 'center' }}>
        <Button onPress={this._activate} title="Activate" />
        <Button onPress={this._deactivate} title="Deactivate" />
      </View>
    );
  }

  _activate = () => {
    /* @info 调用后屏幕将保持常亮，直到调用 <strong>deactivateKeepAwake()</strong>。 */ activateKeepAwake(); /* @end */
    alert('Activated!');
  };

  _deactivate = () => {
    /* @info 停用 KeepAwake；如果它从未被激活，则什么也不做。 */ deactivateKeepAwake(); /* @end */
    alert('Deactivated!');
  };
}
```

</SnackInline>
