HTTP 的结构化字段值，[IETF RFC 8941](https://tools.ietf.org/html/rfc8941)，是一项旨在规范标头语法并促进嵌套数据的提案。

由于它仍处于进行中状态，Expo 维护了一个自定义版本，仅实现了 [IETF RFC 8941](https://tools.ietf.org/html/rfc8941) 中定义的协议的以下子集：

- 所有键值
- 字符串、整数和小数项
- 字典
