Classic Updates 是 Expo 提供的一项服务，它使用 Expo CLI 的 `expo publish` 命令向最终用户分发更新。Expo 一直支持该服务直到 2022 年 8 月，随后它被 [EAS Update](/eas-update) 取代。

Classic Updates 将在 2023 年和 2024 年逐步停止支持。如果你的项目正在积极使用 Classic Updates，请迁移到 [EAS Update](/eas-update/migrate-from-classic-updates)。
