> **重要** **EAS Metadata** 处于 [beta](/more/release-statuses/#beta) 阶段，可能会有破坏性变更。

EAS Metadata 中的商店配置包含了原本需要通过应用商店后台手动提供的信息。
本文档概述了你在商店配置中对象的结构。

> 如果你使用 [VS Code Expo Tools extension](https://github.com/expo/vscode-expo#readme)，你可以在编辑器中通过自动补全、建议和警告获取所有这些信息。

## 配置架构

商店配置对象中的一个关键属性是 `configVersion` 属性。
应用商店可能会要求更多信息，或更改现有信息结构，才能发布你的应用。
此属性有助于对不向后兼容的更改进行版本管理。

EAS Metadata _目前_ 仅支持 Apple App Store。

### Apple advisory

Apple 使用一份复杂的问卷来确定应用的 [年龄分级](https://help.apple.com/app-store-connect/#/dev599d50efb)。
App Store 上的家长控制会使用这个计算出的年龄分级。
默认情况下，EAS Metadata 会对每个问题使用限制最少的答案。

<Collapsible summary="使用限制最少答案的完整 advisory">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "advisory": {
      "alcoholTobaccoOrDrugUseOrReferences": "NONE",
      "contests": "NONE",
      "gamblingSimulated": "NONE",
      "horrorOrFearThemes": "NONE",
      "matureOrSuggestiveThemes": "NONE",
      "medicalOrTreatmentInformation": "NONE",
      "profanityOrCrudeHumor": "NONE",
      "sexualContentGraphicAndNudity": "NONE",
      "sexualContentOrNudity": "NONE",
      "violenceCartoonOrFantasy": "NONE",
      "violenceRealistic": "NONE",
      "violenceRealisticProlongedGraphicOrSadistic": "NONE",
      "gambling": false,
      "unrestrictedWebAccess": false,
      "kidsAgeBand": null,
      "ageRatingOverride": "NONE",
      "koreaAgeRatingOverride": "NONE"
    }
  }
}
```

</Collapsible>

### Apple categories

App Store 通过[将应用分类到各个类别中](https://developer.apple.com/app-store/categories/)，并使用主类别、次类别以及可能的子类别，帮助用户发现新应用。

<Collapsible summary="主类别和次类别">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "categories": ["FINANCE", "NEWS"]
  }
}
```

</Collapsible>

<Collapsible summary="主类别、子类别和次类别">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "categories": [["GAMES", "GAMES_CARD", "GAMES_BOARD"], "ENTERTAINMENT"]
  }
}
```

</Collapsible>

### Apple info

App Store 是一项面向全球用户、支持多种语言的服务。
你可以用[多种语言](#apple-info-languages)本地化你的 App Store 展示信息。

<Collapsible summary="英文（美国）的最小本地化信息">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "info": {
      "en-US": {
        "title": "Awesome app",
        "privacyPolicyUrl": "https://example.com/en/privacy"
      }
    }
  }
}
```

</Collapsible>

<Collapsible summary="用英文（美国）编写的完整本地化信息">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "info": {
      "en-US": {
        "title": "App title",
        "subtitle": "Subtitle for your app",
        "description": "A longer description of what your app does",
        "keywords": ["keyword", "other-keyword"],
        "releaseNotes": "Bug fixes and improved stability",
        "promoText": "Short tagline for your app",
        "marketingUrl": "https://example.com/en",
        "supportUrl": "https://example.com/en/help",
        "privacyPolicyUrl": "https://example.com/en/privacy",
        "privacyChoicesUrl": "https://example.com/en/privacy/choices"
      }
    }
  }
}
```

</Collapsible>

{
  "configVersion": 0,
  "apple": {
    "release": {

### Apple 审核

在 App Store 上发布应用之前，需要先获得商店审核通过。
App Store 审核团队必须拥有测试你的应用所需的全部信息，否则你可能会面临应用被拒。

<Collapsible summary="最少审核信息">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "review": {
      "firstName": "John",
      "lastName": "Doe",
      "email": "john@example.com",
      "phone": "+1 123 456 7890"
    }
  }
}
```

</Collapsible>

<Collapsible summary="完整审核信息">

```json store.config.json
{
  "configVersion": 0,
  "apple": {
    "review": {
      "firstName": "John",
      "lastName": "Doe",
      "email": "john@example.com",
      "phone": "+1 123 456 7890",
      "demoUsername": "john",
      "demoPassword": "applereview",
      "demoRequired": false,
      "notes": "这是一个主要用于教育目的的示例应用。"
    }
  }
}
```

</Collapsible>
